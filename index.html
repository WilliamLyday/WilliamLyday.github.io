HigherLower game

Goal: Write a number guessing game, where a number is chosen between 1 and
100. The user guesses, the game indicates higher, lower, or correct. The user
continues to guess until correct. The score is displayed and then a top 10
high scores is displayed. If this game's score was within the top 10, this
score should be displayed in that list.

Additional Requirements:
We need a typical login (sign up new/existing)
&emsp This entails a db to store user data as well as
&emsp crptographically hashed password for security
Must support SSL/TLS certificates to provide secure connection
